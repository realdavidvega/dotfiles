Apple Music Token Helper

This page helps you get your Music User Token so you can export playlists via the Apple Music API.

Note: You need an Apple Developer account (free) and a MusicKit private key for the full setup. If you do not have these yet, follow the setup guide in the export script first.

Step 1: Authorize

Click the button below to sign in with Apple Music. This will request permission to read your library.

Step 2: Copy Your Token

After signing in, your Music User Token will appear below. Copy it and use it with the export script.

Save for Reuse

Save the token to your config so you do not need to do this every time:

python3 ~/.dotfiles/scripts/mp3-tagger/export_apple_music_api.py \
  --dev-token "YOUR_DEV_TOKEN" \
  --user-token "YOUR_USER_TOKEN" \
  --save-config
